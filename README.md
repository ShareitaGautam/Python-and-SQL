# Python-and-SQL
This repository contains solutions to a Python and SQL coding assessment that demonstrates fundamental programming and database querying skills. The tasks focus on string manipulation, dictionary aggregation, list processing, and SQL queries involving ranking functions and joins.
Python Tasks

String Cleaning & Transformation
Remove vowels from a sentence, replace spaces with _, and convert it to title case.

Dictionary Aggregation
Calculate total sales for each product from a list of sales transactions.

Unique Numbers in List
Return numbers that appear only once in a list.

SQL Tasks

Second Highest Salary
Write a SQL query to find the second highest distinct salary from the employees table.

Customers with No Orders
Write a SQL query to find customers who have not placed any orders.
